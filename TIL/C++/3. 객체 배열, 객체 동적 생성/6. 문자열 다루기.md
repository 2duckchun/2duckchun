# string 클래스

## 문자열 비교

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
	string name = "albis";
	string alias = "presly";
	int res = name.compare(alias);
	if (res == 0) cout << "두 문자열이 같음"; // 두 문자열이 같다.
	else if (res < 0) cout << name << " < " << alias; // name이 앞에 온다.
	else cout << alias << " < " << name << endl; // name이 뒤에 온다.
	return 0;
}
```

- `compare` 함수는 두 문자열이 같으면 0, str보다 사전 순으로 앞에 오면 음수, 뒤에 오면 양수를 리턴한다.
